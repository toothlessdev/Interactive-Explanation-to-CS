CPU Scheduling

FCFS Scheduling

선입선출 방식

First Come First Served

FCFS FIFO 스케쥴링은 자료구조 Queue 와 비슷하게 작동한다.

First Come First Served, 먼저 오는 Job(Task) 에게 CPU 자원을 할당해주는 방식이다

예시

Example

P1 이 도착하고, 이후에 P2, P3 가 도착했다고 가정해보자

먼저 오는 작업 순서대로 P1, P2, P3 가 차례대로 CPU 자원을 받게 된다.

0.00

Idle

장점

Advantages

간단하고 구현이 쉽다

단점

Disadvantages

Covoy Effect

모든 작업이 동일한 TurnaroundTime 을 가지지 않을수도 있다.

가장 먼저 도착한 작업의 TurnaroundTime 이 길면, 평균 TurnaroundTime 이 길어진다.
